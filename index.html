MOD_05 // INTERFACE

The Chat Loop

A standard CS1 exercise: Managing state.


prompt = [{"role": "system", "content": "You are a tutor"}]
while True:
    user_in = input("You: ")
    prompt.append({"role": "user", "content": user_in})
    
    reply = chat(prompt)
    prompt.append({"role": "assistant", "content": reply})
